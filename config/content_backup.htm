建立倉儲

先進去老師倉儲的cmsimde site，複製網址

到Create a new repository點開import a repository

貼上剛剛複製的網址，Repository Name設為cd2022

接著到小黑窗

cd tmp

git clone --recurse-submodules https://github.com/a40923101/cd2022.git

cd cd2022

cms

複製網址 https://127.0.0.1:9443/ 就可以開啟倉儲了。

tokens

到developer setting 中創建一個沒有期限的token並且將下面選項都打勾之後會出現一串文字碼

開啟config到白窗，複製貼上指令然後儲存。接著到小黑窗acp，檢查完成。

Wink影片上傳

影片做完時，會有四個檔案

把隨身碟裡的JavaScript指令檔拉到gogle。

複製起來，開頭增加<script>，結尾增加</script>
如下
<script>
var winkVideoData = {
  dataVersion: 1,
  frameRate: 10,
  buttonFrameLength: 5,
  buttonFrameOffset: 2,
  frameStops: {
  },
};
</script>

接著把隨身碟裡的HTML檔拉到gogle，到自己影片點右鍵再按檢視網頁原始碼。

把下圖框起來的部分複製起來。

接著按save，按generate_pages。

到小黑框打

acp "xxx" (xxx隨便輸入甚麼都可)

推完之後回去github，找downloads資料夾。

進入downloads資料夾後，點選Add File，按Upload files。

按choose your files

選做好的wink影片(mp4檔)

檔案跑完之後，按Commit changes。

到小黑框打

git pull

完成後就可以去倉儲檢查影片有無推成功。

Leo

在小黑窗打

cd cd2022

leo

進到leo，接著點選File裡面的Open-Outline

接著開啟cd2022的config裡面的pelican

修改裡面原有的資料，改成自己的學號和標題

這裡網址也都改成自己倉儲

修改自己要的標題資訊

修改好後對local-blog按下右鍵，在按下Goto Script

在小黑窗打上

cd cd2022 

pelican markdown -o blog -s local_publishconf.py

第一次打會出現一個錯誤，需要再cd2022的markdown裡面加個images的資料夾就可以解決錯誤。

Wink

下載Wink3.0版

下載網址:https://www.debugmode.com/wink/

Wink3.0是一套免費的電腦操作流程影片錄製軟體，與ShareX相同的是，兩套軟體的影片製作核心都是採用ffmpeg.exe.不同的地方在於:Wink3.0沒有開源，可能是採用 C++ 編寫，且Wink3.0可以在主影格中加入文字說明，以及 html 按鈕。比較適合用來製作需要較多文字說明輔助的教學影片。

ShareX

下載網址:https://getsharex.com/downloads/
ShareX是一套採用 C# 編寫的開源套件，適合製作加上搭配語音或說明字幕的操作教學影片。
